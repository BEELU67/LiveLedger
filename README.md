# LiveLedger

## Over het project
LiveLedger is een communityplatform voor concertliefhebbers waarop gebruikers setlists van concerten kunnen uploaden, bekijken en beoordelen. Het platform focust volledig op livemuziek, concertervaringen en community-interactie.

Inspiratie:
- Setlist.fm
- Letterboxd
- Last.fm

---

## Functionaliteiten

### Front-end
- Homepage met recente shows en uitgelichte artiesten
- Artiestpagina's
- Showpagina's met setlists
- Zoekfunctie
- Gebruikersprofielen
- Responsive design

### Back-end
- Registratie en login
- Shows toevoegen
- Reviews en scores
- Reactiesysteem
- Adminpaneel
